# 🥊 CT Don Diaz — Website Comercial para Academia de Lutas

Projeto de website institucional desenvolvido como **case de portfólio front-end**, simulando um cenário real de cliente para uma academia de artes marciais.

O objetivo foi construir uma aplicação web focada em **experiência do usuário, responsividade, identidade visual esportiva e conversão comercial**, aplicando boas práticas modernas de desenvolvimento web sem o uso de frameworks.

---

## 🎯 Objetivo do Projeto

Este projeto demonstra competências em:

* Desenvolvimento de interfaces comerciais reais
* Estruturação semântica com HTML5
* Responsividade avançada
* Manipulação do DOM com JavaScript puro
* Organização visual orientada à conversão
* Experiência do usuário (UX) aplicada a negócios locais

---

## 🧠 Problema Proposto

Academias locais normalmente possuem baixa presença digital ou páginas pouco otimizadas para captação de alunos.

O desafio foi criar um site que:

* transmitisse autoridade e profissionalismo
* funcionasse perfeitamente em dispositivos móveis
* facilitasse o contato imediato via WhatsApp
* apresentasse informações essenciais em poucos segundos

---

## 🛠️ Stack Tecnológica

* **HTML5**
* **CSS3 (Custom Properties + Media Queries)**
* **JavaScript Vanilla (ES6+)**
* **Bootstrap Icons (CDN)**
* **Google Maps Embed**

Sem uso de frameworks ou bibliotecas JS externas, priorizando domínio dos fundamentos.

---

## ⚙️ Funcionalidades Implementadas

### Interface e UX

* Layout mobile-first
* Navegação por âncoras
* Menu responsivo com toggle
* Hierarquia visual focada em conversão

### Interatividade

* Alternância de tema (Light/Dark Mode)
* Persistência de preferências via `localStorage`
* Header inteligente que reage ao scroll
* Carousel automático no hero section

### Responsividade Avançada

Breakpoints adaptados para:

* Mobile
* Tablet
* Desktop
* 2K (2560px)
* 4K (3840px)

Escala tipográfica baseada em `rem` para melhor adaptação.

## 📊 Decisões Técnicas

**JavaScript puro ao invés de frameworks**
→ foco em domínio da base da web antes de abstrações.

**Carousel customizado**
→ redução de dependências externas e maior controle de performance.

**Tema persistente**
→ melhora da experiência do usuário recorrente.

**CTA direto para WhatsApp**
→ estratégia comum em negócios locais para aumentar conversão.
